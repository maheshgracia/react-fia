Empire State Building

  • Empire State Building Tricolor and Diwali Lighting

    New York City's iconic Empire State Building has been illuminated in the colors of the Indian flag to honor India's Independence Day. Since 2014, FIA, in partnership with the Empire State Building, has been organizing this event on an annual basis. To commemorate the Diwali celebration, the Empire State Building is illuminated in orange tones. New York City's thriving Indian community is reflected in this lovely custom, which is part of the Diwali celebrations.

    Attendees include celebrities, dignitaries, and prominent Indian American personalities, symbolizing the legacy of the Indian diaspora. Thousands of people visit this magnificent site annually.
